# Tienda Virtual – Proyecto Académico

Proyecto desarrollado durante el semestre como parte
de la formación en Ingeniería de Sistemas.

## Descripción
Tienda virtual básica desarrollada para practicar
conceptos de desarrollo frontend y estructura de una
aplicación web.

## Tecnologías utilizadas
- React
- HTML
- CSS
- JavaScript

## Funcionalidades
- Visualización de productos
- Interfaz responsive
- Estructura básica de componentes

## Objetivo
- Reforzar conceptos de frontend
- Comprender la estructura de una aplicación en React
- Aplicar estilos y lógica básica con JavaScript

## Estado
Proyecto académico / En formación
